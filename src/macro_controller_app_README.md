# 宏控制桌面应用说明

整合「宏播放」与「键盘控制器」的 Python 桌面程序，支持从指定步开始播放、暂停、手动控制后继续。

## 运行方式

在 `src` 目录下执行：

```bash
cd d:\work\app\pytouch\src
python macro_controller_app.py
```

或确保 `src` 在 Python 路径中后运行：

```bash
python macro_controller_app.py
```

## 功能概览

| 功能 | 说明 |
|------|------|
| 打开宏文件 | 选择宏 JSON 文件（如 `macros/macro_20260224_220403.json`） |
| 步骤列表 | 显示所有步骤，绿色=已执行，黄色=当前，灰色=待执行 |
| 从第 N 步开始 | 输入步数或双击某一步，再点「开始/继续播放」即从该步执行 |
| 暂停 | 当前步执行完后暂停并释放串口 |
| 手动控制 | 弹出窗口，输入 1–16 / `@` / 其它串口指令，用法同键盘控制器 |
| 停止 | 彻底终止本次播放 |

## 典型流程

1. **打开宏文件** → 步骤列表加载，底部显示「共 N 步」。
2. **（可选）设置起始步**：在「从第 __ 步开始」填数字，或**双击步骤列表中某一行**，自动设为从该步开始。
3. 点击 **「开始/继续播放」**，宏按顺序执行，列表实时更新已执行/当前步。
4. 若中途需要手动操作：
   - 点击 **「暂停」**，等当前步执行完；
   - 点击 **「手动控制」**，在弹出窗口中输入指令（如数字 1–16、`@` 等），用完后关闭窗口；
   - 主界面「从第 __ 步开始」会自动变为「下一步」，点击 **「开始/继续播放」** 即可从该步继续。
5. 若要完全结束本次播放，点击 **「停止」**。

## 界面说明

- **串口**：默认 `COM5`，可按实际串口修改。
- **从第 __ 步开始**：1 表示从第 1 步开始，暂停后会自动变为「下一步」的步号。
- **步骤列表**：每行对应 JSON 中一条（说明或 pin+interval），双击可设为起始步。
- **状态/进度**：底部显示当前状态和「已执行 X 步，当前第 Y / Z 步」。
